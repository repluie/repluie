Cityhub Operation Platform

Cityhub Operation Platform은 데이터 수집, 모니터링, 제어, 분석, 액션 처리를
효과적으로 제공하는 운영 플랫폼입니다.

SOLUTIONS

Smart Enterprise

Smart Connect로부터 수집된 데이터를 통합하여 제어/감시하는 엔진입니다. 분산처리 기능을 활용하여 단독 빌딩 뿐 아니라 여러 개 빌딩을 하나의 중앙센터에서 관리 할 수 있도록 구축함으로써 관리의 효율성을 향상시킬 수 있으며, 빌딩 뿐 아니라 공장 등의 서비스에도 적용하여 대량의 제어 감시 시스템을 구축할 수 있습니다.

Smart Connect

디바이스로부터 관제점의 값을 모으고 제어하는 일종의 I/O 서버로써 다양한 디바이스들과 통신을 할 수 있습니다. 80종 이상의 표준 및 비 표준 프로토콜을 지원함으로써 디바이스 선택의 폭을 넓게 합니다. 또한 다양한 외부 시스템 연계에 필요한 커넥터를 제공합니다.

Smart Builder

통합관제 실현을 위한 엔지니어링 도구의 일종으로 Device와 프로토콜 Profile 데이터 엔지니어링 및 화면 제작을 위한 저작 도구와 모니터링을 위한 Runtime 뷰어로 구성되어 있습니다.

어디서나 접근 가능

Web 기반으로 구현된 플랫폼으로써 어디서나 사용이 가능하며, 모바일 기기를 통한 운영이 가능합니다.

대규모 처리 성능, 수평 확장 지원

Cityhub building 하나의 Node 당 300,000 Point 처리 성능을 지원합니다. 또한 Scale Out 지원으로 대규모 사이트 및 서비스 연속성 측면에서 안정적인 서비스 지원이 가능합니다.
(Scale Out, 자체 Fault-Tolerance, 원격감시, Platform-Platform 지원)

N point 운영 뷰어

사용자의 역할별 다양한 업무 형태에 따라 선택적으로 사용할 수 있는 다양한 운영 뷰어를 제공함으로써 업무효율을 높일 수 있도록 지원합니다. (HMI 운영 뷰어, Stream-Analytics 뷰어, Dashboard 뷰어)

저렴한 구축 비용

별도 개발 없이 간단한 엔지니어링만으로 감시/제어 시스템 구축이 가능하고 Ready-Made된 운영뷰어 제공으로 효율성을 높일 수 있으며, 오픈소스 S/W 제품 지원으로 구축 비용을 절감할 수 있습니다.

Big-Data Bridge

No-SQL 기반의 History Data 관리로 Big-Data 기술과 결합을 손쉽게 지원합니다.

통합된 업무처리 절차

제어, 모니터링, 경보처리, SOP(Standard Operation Procedure) 등 통합 된 업무처리를 지원합니다.

다양한 프로토콜 Adapter

10여종의 표준, 비표준 디바이스 Adapter를 보유하고 있습니다.

80+ Reference

단독빌딩, 복합빌딩, 원격감시, 군감리, 공장, 전력, 에너지 등 다양한 도메인 영역에서 80개 이상의 Reference를 보유하고 있습니다.

3-way 확장이 가능한 플랫폼

시장의 Needs에 따라 3가지 영역에서 손쉽게 확장할 수 있는 기능을 제공하고 있습니다.

Any Device

Easy to Connect

Smart Connect를 통해 다양한 디바이스 별 어댑터를 지원합니다. 새로운 프로토콜이나 디바이스 추가 시 별도의 서비스 수정 없이 어댑터만 추가하여 사용할 수 있습니다.

Any Domain

Easy to Expand

빌딩 뿐 아니라 데이터센터, 공장, 에너지, 도메인 등 다양한 디바이스가 산재되어 있는 사업 영역에서도 적용 가능합니다.

Any Service

Easy to Build

시장의 Needs에 따라 고객에 필요한 서비스를 추가하기 용이하여 사용자 요구사항에 맞는 최적화된 시스템을 편리하게 구현할 수 있습니다.

다양한 방식의 시스템 구성 지원

고객의 업무 특성에 맞는 다양한 시스템 구성을 지원함으로써 고객의 운영 환경에 최적화된 시스템 구성이 가능합니다.

혁신적인 데이터 처리 효율

SGP-RLE-Data(Run Learning Encoding / Decoding for Data) 알고리즘을 적용하여 데이터 압축 저장/추출 관리함으로써 대량의 이력 데이터도 효율적으로 처리합니다.
SGP-RLE_Tx(Run Learning Encoding / Decoding for Transaction) 프로세싱 알고리즘을 적용하여 Things 데이터 처리 측면에서 혁신적인 처리 효율을 제공합니다.

웹 기반 통합 운영 플랫폼

Device Profiling - 데이터 외부 전파까지 End-To-End 운영이 가능합니다.
대용량 관제 데이터 처리 및 사용자 역할에 최적화 된 운영 뷰어를 제공합니다.
